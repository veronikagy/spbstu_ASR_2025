Диаграмма Использования (Дипсик):
Тема научной работы:
«Разработка подхода к сегментации пользователей на основе анализа их поведения и эмоциональных откликов для повышения эффективности CRM-систем».
Информационная система для проектирования:
Система анализа клиентского опыта (Customer Experience Analysis System)
Действующие лица:
•	Клиент — физическое лицо, взаимодействующее с компанией через различные каналы (сайт, приложение, кол-центр). Является источником данных о поведении и отзывах.
•	Аналитик данных — специалист, который настраивает модель сегментации, запрашивает отчеты и анализирует полученные сегменты и закономерности.
•	Менеджер по клиентскому опыту — использует систему для получения готовых инсайтов о клиентских сегментах для принятия решений по улучшению сервиса и персонализации коммуникаций.
Администратор системы — отвечает за управление учетными записями, правами доступа и интеграцию системы с источниками данных (например, CRM, базы отзывов, системы веб-аналитики).
Сгенерируй PlantUML код для диаграммы вариантов использования системы анализа клиентского опыта. Укажи действующие лица. Например, для  библиотеки: читатель, библиотекарь, администратор. Перечисли основные случаи использования. Например, для  библиотеки: такие как просмотр каталога, бронирование книг, продление срока пользования, добавление в избранное, выдача книг, принятие возврата, формирование отчёта, добавление книги в каталог, удаление книги из каталога, и др.
