Создай профессиональную диаграмму вариантов использования (Use Case) в PlantUML для системы онлайн-
кинотеатра (стримингового сервиса).
Акторы:
1. Гость — неавторизованный посетитель
2. Пользователь — наследует от Гостя, имеет подписку
3. Премиум-пользователь — наследует от Пользователя, расширенные возможности
4. Администратор контента — управляет каталогом
5. Модератор — работает с отзывами
6. Платёжная система — внешний актор
Варианты использования с отношениями:
- Просмотр каталога (базовый)
- Поиск фильмов (include: фильтрация по жанру)
- Регистрация (extend: подтверждение email)
- Авторизация (extend: двухфакторная аутентификация)
- Просмотр фильма (include: проверка подписки)
- Просмотр в 4K (extend от Просмотр фильма, только для Премиум)
- Скачивание для офлайн (extend, только Премиум)
- Оформление подписки (include: оплата)
- Оплата (связь с Платёжной системой)
- Добавление в избранное
- Оценка и отзыв
- Управление каталогом (админ)
- Модерация отзывов (модератор)
Технические требования:
- Добавь границу системы с названием "Онлайн-кинотеатр StreamFlix"
- Используй цветовое кодирование для разных типов акторов
- Добавь заметки (notes) к ключевым use case
- Группируй связанные варианты использования
- Соблюдай стандарт UML 2.0
- Добавь заголовок и легенду
